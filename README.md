## Building

- For Windows

```sh
dx build -r --windows
```

- For Android (No sign)

```sh
# Set up JDK, Android SDK, Android NDK 28.0.12433566
# rustup target install aarch64-linux-android
dx build -r --android --target aarch64-linux-android
```
